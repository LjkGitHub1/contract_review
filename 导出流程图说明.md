# 导出流程图为PNG说明

## 方法一：使用Playwright（推荐）

### 1. 安装依赖
```bash
pip install playwright
playwright install chromium
```

### 2. 运行导出脚本
```bash
python export_flowcharts_simple.py
```

### 3. 输出文件
- `合同流程图_合同起草流程图.png`
- `合同流程图_合同审核流程图.png`

## 方法二：使用Selenium

### 1. 安装依赖
```bash
pip install selenium pillow
```

### 2. 安装ChromeDriver
- 下载ChromeDriver：https://chromedriver.chromium.org/
- 将ChromeDriver添加到系统PATH，或放在项目目录下

### 3. 运行导出脚本
```bash
python export_flowcharts_to_png.py
```

## 方法三：手动导出（最简单）

### 使用浏览器截图

1. **打开HTML文件**
   - 在浏览器中打开 `合同流程图.html`

2. **截图合同起草流程图**
   - 滚动到"一、合同起草流程图"部分
   - 使用浏览器开发者工具（F12）选择流程图区域
   - 或使用截图工具（如Windows的Snipping Tool）截图
   - 保存为 `合同流程图_合同起草流程图.png`

3. **截图合同审核流程图**
   - 滚动到"二、合同审核流程图"部分
   - 使用相同方法截图
   - 保存为 `合同流程图_合同审核流程图.png`

### 使用浏览器打印功能

1. 打开 `合同流程图.html`
2. 按 `Ctrl+P` 打开打印对话框
3. 选择"另存为PDF"或使用打印到PDF功能
4. 然后使用PDF转PNG工具转换

## 方法四：在线工具

可以使用在线SVG转PNG工具：
1. 打开 `合同流程图.html`
2. 复制SVG代码
3. 使用在线工具转换：https://svgtopng.com/

---

**推荐使用方法一（Playwright）**，最简单且效果最好。
